# ParallelAI: Multi-Provider AI Security Tool

## Overview
ParallelAI is a command-line tool that queries multiple AI providers (Groq, Together AI, OpenRouter) with intelligent fallback, designed specifically for cybersecurity research and analysis.

## Research Applications
This tool powers the experiments documented in our [ParallelAI Research Edition](https://github.com/jasonclarkagain/parallelai-research) repository, including:
- Comparative malware analysis (Mirai vs Zeus botnets)
- SQL injection technique classification  
- Critical infrastructure vulnerability assessment
- Phishing detection methodology research
